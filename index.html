Introduction

It was first released in 1991 by Guido van Rossum, and has since become one of the most popular programming languages in the world, with a vast and growing community of users and developers.

Python is known for its simplicity and readability, making it easy to write and understand code. This is achieved through the use of clear and concise syntax, as well as a focus on readability and ease of use. Python also has a large number of libraries and modules that provide a wide range of functionality, making it a flexible and powerful language for a variety of tasks, including web development, data analysis, scientific computing, and much more

Whether you are new to programming or an experienced developer looking to learn Python, this documentation is a valuable resource to help you get started and grow your skills.

Variables and Types

In Python, variables are used to store data, and the type of data that a variable can store is determined by its data type. Python has several built-in data types, including:

  • Integers (int): whole numbers, such as 1, 2, and 3.
  • Floating-point numbers (float): numbers with decimal places, such as 3.14, 2.71828, and 0.5.
  • Strings (str): sequences of characters, such as "Hello, World!" or "Alice".

To assign a value to a variable in Python, you can use the assignment operator (=). For example:

x = 42
y = 3.14
name = "Alice"

In this example, the integer value 42 is assigned to the variable x, the floating-point value 3.14 is assigned to the variable y, and the string value "Alice" is assigned to the variable name.

It's important to note that in Python, variable names cannot start with a number, and should be descriptive of the data they store. Additionally, variable names are case-sensitive, so x and X are considered to be different variables.

In addition to the built-in data types, Python also provides several other data structures for storing and organizing data, such as lists, tuples, and dictionaries, which are covered in more detail in later sections of this documentation

Lists tuples and dictionaries

In Python, there are several built-in data structures for storing and organizing data. These include:

  • Lists

    A list is a collection of values that can be of any data type, including other lists. Lists are created using square brackets [] and values are separated by commas. For example:

    fruits = ["apple", "banana", "cherry"]

    In this example, fruits is a list containing the strings "apple", "banana", and "cherry". Lists are ordered, meaning that the order of elements in a list is preserved.

    Lists are mutable, meaning that you can change the values of elements in a list after it has been created. For example:

    fruits[1] = "orange"
    print(fruits)

    This code changes the second element of the fruits list (at index 1) from "banana" to "orange". The updated list is then printed to the screen, which would output ['apple', 'orange', 'cherry'].

  • Tuples

    A tuple is similar to a list, but is immutable, meaning that the values of elements in a tuple cannot be changed after it has been created. Tuples are created using parentheses () and values are separated by commas. For example:

    person = ("Alice", 30, "female")

    In this example, person is a tuple containing the values "Alice", 30, and "female".

  • Dictionaries

    A dictionary is a collection of key-value pairs. Each key maps to a value. Dictionaries are created using curly braces {} and keys and values are separated by colons. For example:

    person = {"name": "Alice", "age": 30, "gender": "female"}

    this example, person is a dictionary with keys "name", "age", and "gender", which map to the values "Alice", 30, and "female", respectively.

    Dictionaries are also mutable, meaning that you can add, change, or delete key-value pairs after a dictionary has been created. For example:

    person["address"] = "123 Main St."
    print(person)

    This code adds a new key-value pair to the person dictionary, with key "address" and value "123 Main St.". The updated dictionary is then printed to the screen, which would output {'name': 'Alice', 'age': 30, 'gender': 'female', 'address': '123 Main St.'}.

    These data structures provide different ways to store and organize data, and you can use the one that is most appropriate for your needs. In later sections, you will learn how to manipulate and work with lists, tuples, and dictionaries.

Basic Operators

Python provides a number of basic arithmetic operators that can be used to perform mathematical operations on numbers and variables. These operators include addition (+), subtraction (-), multiplication (*), and division (/). The following example demonstrates how to use these operators in Python:

x = 42
y = 7
print(x + y) # 49
print(x - y) # 35
print(x * y) # 294
print(x / y) # 6.0

In addition to these arithmetic operators, Python also provides other types of operators such as the modulus operator (%), which returns the remainder of a division operation, and the power operator (**), which raises a number to a specified power.

print(x % y) # 0
print(x ** y) # 1419857

It's also possible to use comparison operators (>, >=, <, <=, ==, !=) in Python to compare values and variables. These operators return a Boolean value (True or False) depending on the result of the comparison.

print(x > y) # True
print(x < y) # False
print(x == y) # False
print(x != y) # True

By combining these operators, you can perform complex mathematical operations and make decisions in your code based on the results.

String Formatting

In Python, you can use string formatting to insert values into strings. String formatting allows you to create dynamic strings that can be adjusted based on the values of variables or other inputs. The format method is one of the most commonly used string formatting techniques in Python.

The format method works by replacing placeholders in a string with values. Placeholders are represented by curly braces {} and can contain positional or keyword arguments that specify how the values should be inserted into the string.

Here's an example that demonstrates how to use the format method to insert a value into a string:

name = "Alice"
print("Hello, {}!".format(name)) # Hello, Alice!

In the example above, the string "Hello, {}!" contains a placeholder {}. The format method is used to replace the placeholder with the value of the name variable. The result of the format method is a new string that contains the value of the name variable.

You can also use the f-string syntax (introduced in Python 3.6) to insert values into strings. An f-string is a string literal that is prefixed with the letter f. Within an f-string, expressions can be evaluated and included inside the string literal, by enclosing them in curly braces {}.

name = "Alice"
print(f"Hello, {name}!") # Hello, Alice!

String formatting is a powerful tool for creating dynamic strings in Python and can be used for a variety of tasks, such as creating messages, generating URLs, or constructing SQL queries.

Basic String Operations

Python provides a wide range of built-in functions and methods for working with strings. With these tools, you can perform various operations on strings, such as concatenation, repetition, and character access.

Concatenation: You can concatenate two or more strings using the addition (+) operator. For example:

first_name = "Alice"
last_name = "Smith"
full_name = first_name + " " + last_name
print(full_name) # Alice Smith

In the example above, the first name and last name are concatenated to form the full name

Repetition: You can repeat a string multiple times using the repetition (*) operator. For example:

print("Hello " * 3) # Hello Hello Hello

In the example above, the string "Hello " is repeated three times.

Length: You can find the length of a string using the len function. The length of a string is the number of characters in the string. For example:

full_name = "Alice Smith"
print(len(full_name)) # 9

Character Access: You can access individual characters in a string using indexing. Indexing starts at 0 for the first character in the string. For example:

full_name = "Alice Smith"
print(full_name[0]) # A

In the example above, the first character in the full_name string is accessed using indexing.

Basic string operations are an essential part of working with strings in Python and are used in many different applications. Whether you're working with string data from a file, user input, or a network, understanding how to manipulate and work with strings is a critical skill for any Python programmer.

Conditions

You can use conditions in Python to make decisions based on the values of variables. The if statement is used to specify conditions. he if statement allows you to test a condition and execute a block of code if the condition is true. In the example above, the condition x > 0 is evaluated, and if it is true (in this case, it is), the message "Positive" is printed.

In addition to the if statement, you can use the elif (else if) and else statements to specify additional conditions and the code to execute if those conditions are true:

x = 42
if x > 0:
print("Positive")
elif x < 0:
print("Negative")
else:
print("Zero")

n this example, if x is greater than 0, the message "Positive" is printed. If x is less than 0, the message "Negative" is printed. If neither of these conditions are true (in this case, x is equal to 0), the message "Zero" is printed.

Functions

Functions are reusable blocks of code that can be executed when they are called. Functions are defined using the def keyword and can accept arguments, perform a specific task, and return a value if needed. For example:

def greet(name):
print("Hello, {}!".format(name))
greet("Alice") # Hello, Alice!

In this example, the function greet is defined to accept a single argument name and print a message with that name. The function is then called with the argument "Alice"

Functions can also return a value using the returnkeyword

def add(a, b):
return a + b
result = add(40, 2)
print(result) # 42

In this example, the function add accepts two arguments a and b and returns their sum. The function is called with the arguments 40 and 2, and the returned result is stored in the variable result.

Classes

Classes in Python are a blueprint for creating objects. They allow you to define the structure and behavior of objects in a program, and are an essential aspect of object-oriented programming. A class defines a new data type, and objects of that type can be created by calling the class like a function.

To create a class in Python, use the class keyword, followed by the name of the class and a colon. The class body is indented, and typically contains class variables and methods:

                class Dog:
                    species = "Canis familiaris"
                
                    def __init__(self, name, breed):
                        self.name = name
                        self.breed = breed
                
                    def bark(self):
                        print("Woof!")

In this example, we define a class Dog that has two class variables (species and breed) and one method (bark). The __init__ method is a special method called a constructor, and it is automatically called when an object of the class is created.

To create an object of a class, you simply call the class like a function and pass any required arguments:

dog = Dog("Fido", "Labrador") print(dog.name) # Fido print(dog.breed) # Labrador dog.bark() # Woof!

Classes are a powerful tool for organizing and structuring code, and are used in many different types of programs. They allow you to create objects that encapsulate data and behavior, making it easier to reuse code and build more complex systems

Objects

Objects are instances of classes, and they are one of the core building blocks of object-oriented programming in Python. A class is a blueprint that defines the attributes and behavior of a particular type of object. For example:

                class Dog: 

                    def __init__(self, name, breed): 

                        self.name = name 

                        self.breed = breed 

                    
                    def bark(self):  

                        print("Woof!") 

                
                my_dog = Dog("Max", "Labrador") 

                print(my_dog.name) # Max  

                print(my_dog.breed) # Labrador  

                my_dog.bark() # Woof!

In this example, the Dog class is defined with two attributes name and breed and a method bark. An instance of the class Dog is created with the name "Max" and the breed "Labrador". The attributes of the instance can be accessed using dot notation, and the method can be called using parentheses

Object-oriented programming in Python allows you to create complex, reusable code by defining and using objects.
